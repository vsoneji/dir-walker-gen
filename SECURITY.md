# Security Policy

## Supported Versions

Security fixes are provided for the latest release on the master branch.

## Reporting a Vulnerability

Please report vulnerabilities privately by opening a GitHub security advisory:
https://github.com/vsoneji/dir-walker-gen/security/advisories/new

If you cannot use GitHub advisories, open an issue and include only non-sensitive details while requesting a private follow-up channel.

## Response Expectations

- Initial acknowledgment target: 3 business days
- Status update target: 7 business days
- Fix timeline: depends on severity and complexity
